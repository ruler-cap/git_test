# Perceptive MPC 论文解读：任务与创新点

**论文**：*Perceptive Model Predictive Control for Continuous Mobile Manipulation*  
**作者**：Johannes Pankert, Marco Hutter  
**发表**：IEEE Robotics and Automation Letters, 5(4): 6177-6184，2020-10  
**DOI**：[10.1109/LRA.2020.3010721](https://doi.org/10.1109/LRA.2020.3010721)  
**代码**：[leggedrobotics/perceptive_mpc](https://github.com/leggedrobotics/perceptive_mpc)

## 工作目标

论文解决的是**连续移动操作**，即机器人不应先停到一个固定工位再操作，而是让轮式底盘和机械臂协同移动，使末端持续跟踪任务空间轨迹。

面向两类任务：

1. **非接触连续作业**：喷漆、喷涂、视觉巡检等。机器人沿墙或工件持续移动，末端保持给定的位置和姿态轨迹。
2. **接触连续作业**：清洁、打磨、抹灰、推拉等。机器人在连续运动中保持目标接触力/力矩，同时不丢失末端路径精度。

目标平台是带机械臂的非完整轮式底盘，包括差速、履带和 skid-steer 类移动底盘；实验主要使用差速底盘移动操作臂 MabiMobile，并在 Swaco 平台上验证可移植性。

## 论文中的具体任务

| 任务 | 具体要求 | 体现能力 |
| --- | --- | --- |
| 白板喷涂 | 机械臂末端带喷枪，沿 1.5 m 参考轨迹连续绘制图案；底盘移动以扩大作业范围。 | 全身协同的连续 6D 末端轨迹跟踪。 |
| 白板接触擦拭 | 海绵工具接触白板，沿墙面移动，同时维持恒定法向力。 | 位姿目标与交互力目标同时存在时的柔顺控制。 |
| 力阶跃跟踪 | 末端保持位置参考，目标法向力按 10 N 阶跃增加。 | 以位移式导纳控制跟踪接触力，无需关节力矩控制接口。 |
| 开工业柜门 | 用 Robotiq 夹爪操作柜门把手；存在把手位置、抓取位姿、状态估计误差。 | 接触不确定性下的导纳顺从和在线全身重规划。 |
| 重物抓取稳定性 | 机器人前伸抓取重物，外力导致倾覆风险。 | MPC 预见未来外力，提前调整底盘/机械臂构型，使 ZMP 保持在支撑多边形内。 |
| MBZIRC 实机应用 | Swaco 平台完成砌砖墙、灭火等任务。 | 同一控制框架对不同轮式底盘、不同机械臂的复用性。 |

## 核心创新点

### 1. 连续全身 SLQ-MPC，而不是“导航后操作”

控制器以底盘位姿和机械臂关节位置为统一状态，以底盘前向速度、转向角速度及各关节速度为统一输入。在预测时域内直接最小化末端位置/姿态误差、控制代价和约束违反代价。

因此底盘和机械臂不是由两个独立控制器按顺序工作，而是在同一个最优控制问题中分配运动。这使末端可以在底盘移动时跨越机械臂静态工作空间边界。

底层采用连续时间 **Sequential Linear Quadratic MPC (SLQ-MPC)**：对非线性系统前向 rollout，在当前轨迹附近线性化动力学、二次近似代价，再解 Riccati 方程得到时变状态反馈和前馈控制。论文使用 2 s 预测时域、20 Hz 优化更新，并以 250 Hz 根据最新状态执行反馈策略。

### 2. 将在线视觉地图真正接入 MPC 的碰撞约束

机器人由多个碰撞球近似。每个预测状态下，MPC 查询碰撞球到障碍物的 ESDF 距离，并用 relaxed barrier function 将避障写进代价。

创新点在于作者修改 Voxblox：在每个体素缓存 ESDF 梯度，查询时用局部线性距离模型替代反复插值和数值差分。这样在线深度相机建图产生的 ESDF 才能在 MPC 的大量预测查询中足够快地使用。

结果是：系统无需预先给出完整 CAD 环境，即可对未知或变化环境进行局部、反应式全身避障。

### 3. 通过任务空间预测导纳实现大负载机器人的力控

高载荷工业机械臂通常只有位置/速度接口，无法直接实现力矩级阻抗控制。论文用**任务空间导纳**解决这一问题：

1. 从力/力矩传感器获得当前接触 wrench。
2. 根据期望 wrench 与测量 wrench 的误差，计算对末端参考轨迹的位置/姿态偏移。
3. 将偏移后的参考轨迹送给全身 MPC。

因此，力控不是附加在末端的独立回路，而是改变 MPC 所跟踪的末端轨迹。控制器可以在力目标、末端路径、碰撞约束和关节可行性之间做预测性的协调。

在白板实验中，30 N 法向力的稳态实测均值为 **29.91 N**，标准差 **0.42 N**；另一组约 55 N 测试的均值为 **54.995 N**，标准差 **0.369 N**。

### 4. 将机械稳定性作为全身预测约束

针对抓重物或大接触力任务，论文根据重力、末端预测交互 wrench、质心和末端位置计算 ZMP，并要求 ZMP 位于支撑多边形的内接圆内。

这个约束使 MPC 在接触或抓取发生前主动移动构型和质心，而不只是外力出现后再纠偏。实验中不加该约束时，机器人前伸抓取后会倾倒；预测时域达到 2 s 后，机器人能提前准备并保持稳定。

### 5. 用软约束增强在线鲁棒性

SLQ 不直接处理硬不等式约束。论文采用 relaxed barrier functions，将以下约束转成可微软代价：

- 环境碰撞和自碰撞。
- 关节位置限制。
- ZMP 稳定性限制。

同时，末端位姿跟踪采用二次代价，而不是绝对等式约束。这样在障碍、接触或状态估计扰动造成任务冲突时，优化问题仍保持可解，控制器会优先生成平滑且安全的折中运动。

## 任务与创新的对应关系

| 任务痛点 | 对应技术 | 解决方式 |
| --- | --- | --- |
| 底盘移动会破坏末端轨迹 | 统一 SLQ-MPC | 联合优化底盘与机械臂，使两者共同补偿末端误差。 |
| 现场障碍未知或变化 | 在线 ESDF + 缓存梯度 | 从深度相机增量建图，并将距离/梯度高频传给 MPC。 |
| 大负载臂没有力矩控制接口 | 预测任务空间导纳 | 将力误差转换为末端轨迹偏移，再由 MPC 生成全身补偿动作。 |
| 重物或大接触力会使底盘倾覆 | ZMP 预测约束 | 在预测期内主动调整质心和末端构型，维持支撑稳定性。 |
| 碰撞、轨迹、力和关节限位相互冲突 | Relaxed barriers + 软轨迹代价 | 避免在线优化不可行，同时保留安全优先的连续退化行为。 |

## 与 `Keep it Upright` 的关键区别

两者都让底盘与机械臂协同参与预测控制，但优化对象不同：

- `Keep it Upright` 重点是托盘上未抓取物体的摩擦、滑移和倾覆约束，以及动态障碍规避。
- 本文重点是连续末端作业的位姿精度、接触 wrench、在线环境感知和底盘机械稳定性。

如果目标是“移动中喷涂、清洁、打磨、擦拭、开门、装配接触”等高精度末端任务，本文的任务空间导纳、在线 ESDF 和 ZMP 约束比 `Keep it Upright` 更直接；若目标是无抓取运输，则仍需保留后者的物体稳定建模。
